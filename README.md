## Built with multiple AIs, end to end

This iOS app was built **entirely with multiple AI coding assistants**.

- When I started this project, I had **zero iOS / SwiftUI app development experience**.
- Requirements, architecture design, SwiftUI code, refactoring, bug fixes, and documentation were all generated by AI based on my prompts and feedback.
- **Every line of SwiftUI UI code in this repository was generated by AI.** No manual UI building.
- I did **not hand-write or copy-paste a single line of SwiftUI UI code** into this project. All code was inserted / updated directly by AI tools.
- My role was to come up with the product idea, chat with AIs, review the code, run builds, test on devices, and iterate through feedback.

In other words:  
**No hand-written SwiftUI. No copy-paste. No manual layout tweaking. Fully AI-driven development.**
